# Wordpress en Docker-Compose🐳

Este Docker Compose fue realizado con las siguientes tecnologías:
* MySql🐬
* Wordpress

## Pasos

```sh
git clone https://github.com/JhonnyVeraCervantes/Wordpress-Docker-Compose.git
```
```sh
cd Wordpress-Docker-Compose
```
```sh
docker-compose up -d
```

## Para visualizar es:
* localhost:8080
